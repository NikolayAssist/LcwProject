# LcwProject
Projede bir tane web servis bir tane de proje core dosyası mevcuttur. proje core dosyasının içindeki apiyi ayağa kaldırmak için publish yapılan hostingin 'base' adresini yazılmalı bu dosyada core içinde ApiInfo/ApiInfo.txt dosyasına yazılmalıdır. 
api projesinde web.config içerisinde connection bilgisi : data source=.;initial catalog=LcwDB;user id=sa;password=sa1234 şeklindedir.
bak dosyası yüklenmiştir.
